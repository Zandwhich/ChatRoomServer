# ChatRoom — Server

The server-side of a Java chat room application. Manages client connections and broadcasts messages to all connected
users.

> **See also:** [ChatRoom Client](https://github.com/Zandwhich/ChatRoomClient)

## Overview

Built in Java, the server listens for incoming socket connections and relays messages between clients. Outgoing messages
include per-user colour data for display in the client UI.
